Yarrow Channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover yo.yarrowdoc.org
Listing available packages: 
pear remote-list -c yarrow
Installing a package: 
pear install yarrow/package_name
Installing a specific version/stability: 
pear install yarrow/package_name-1.0.0
pear install yarrow/package_name-beta
Receiving updates via a feed: 
http://yo.yarrowdoc.org/feed.xml

Yarrow - Simple Documentation Generator
